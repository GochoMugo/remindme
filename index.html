remindme

Don't let yourself forget...

A small Command Line Application that lets you store and retrieve short notes quick

Installation

To install remindme on to your commandline: 

$ pip install remindme

If it fails to install, maybe your require some sudo powers. 

$ sudo pip install remindme

Upgrading to the newest version is as simple as: 

$ sudo pip install --upgrade remindme

Usage

$ remindme -h
...
$ remindme --help

This will give you the usage information on your terminal.

Adding a new remindme is simple. 

$ remindme -a keyword you remember
Enter what you remember now:

> I must not forget the formula for Water.
>
> But if I do, please tell me about it.
> It is H20. Right?
> :end
Note: When you are finished editing the remindme, you need to type :end in a newline to exit. You might also use Cmd/Ctrl + C (keyboard interrupt) to exit editing.

Removing a remindme: 

$ remindme -r keyword you remember
Removing ALL remindmes: (Please be careful here...) 
$ remindme -Ra

So I know you might end up forgetting even the keywords:

$ remindme -l
...
$ remindme --list

This show you a list of all the remindme keywords you have stored.
